  ---
  Step 1 — Install Python 3.11+

  1. Go to https://python.org/downloads → download Python 3.11 or 3.12
  2. Run the installer — check "Add python.exe to PATH" before clicking Install
  3. Verify in a new PowerShell window:
  python --version

  ---
  Step 2 — Install Tesseract OCR (for your PNG images)

  1. Download the installer from: https://github.com/UB-Mannheim/tesseract/wiki
  → get tesseract-ocr-w64-setup-5.x.x.exe
  2. Install it — default path is C:\Program Files\Tesseract-OCR\
  (the code already has this path hardcoded in ingestion/image_parser.py)
  3. Verify:
  & "C:\Program Files\Tesseract-OCR\tesseract.exe" --version

  ---
  Step 3 — Install Ollama

  1. Download from https://ollama.com/download/windows and install
  2. It runs as a background service automatically after install
  3. Verify:
  ollama --version

  ---
  Step 4 — Pull the Required Models

  Run these in PowerShell (downloads ~1.5GB + ~300MB — one-time only):
  ollama pull llama3.2
  ollama pull nomic-embed-text

  Confirm both are ready:
  ollama list

  ---
  Step 5 — Set Up Python Virtual Environment

  Open PowerShell and navigate to the project:
  cd C:\UOTM\RAG\kt-rag
  python -m venv venv
  venv\Scripts\activate

  Your prompt should now show (venv).

  ---
  Step 6 — Install Python Dependencies

  pip install -r requirements.txt

  This installs LangChain, ChromaDB, Chainlit, PyMuPDF, openpyxl, pytesseract, etc.

  ---
  Step 7 — Run Document Ingestion

  python scripts\ingest.py

  Expected output:
  Step 1/3: Loading documents...
    Loading: LeadProductList Manager.pdf
    Loading: MFA_Application_Server_APP.pdf
    ...
    Loading: Anvit's KTSheet (1).xlsx
    Loading: UOTM_PRODUCTION_SERVER_SETUP.png
    ...
  ✅ Loaded X document sections

  Step 2/3: Chunking documents...
  ✅ Created X chunks

  Step 3/3: Embedding & storing...
  ✅ Stored X chunks in vectorstore

  The first run embeds everything — may take a few minutes. A vectorstore/ folder will be created.        

  ---
  Step 8 — Test with a CLI Query

  python scripts\query.py "What is the MFA application server setup?"

  If you get an answer with source citations, the pipeline is working.

  ---
  Step 9 — Launch the Chat UI

  chainlit run ui\app.py --host 0.0.0.0 --port 8000

  Then open your browser at: http://localhost:8000
